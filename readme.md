# Boss Battle

#### Controls

Moving is wasd

Block is right mouse button

Attack is left mouse button

Space is dodge

Jump is F 

If you block at the right time, you can perry some attacks

### About
Boss battle is a game where you battle a boss, with the aim to beat them.
